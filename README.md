SkillBridge is a web-based platform designed to connect learners, job seekers, and skill providers in a unified ecosystem. The system aims to bridge the gap between academic knowledge and industry requirements by enabling users to develop skills, showcase their abilities, and access relevant opportunities.

Technical Stack

Backend: PHP 8.x

Database: MySQL

Frontend: HTML5, CSS3, Vanilla JavaScript

Libraries: (Add if you used any, e.g., Chart.js, Bootstrap, etc.)

Environment: Optimized for XAMPP / Apache

Core Features

User Authentication System
Secure registration and login for students and skill providers.

Skill Development Module
Users can explore, enroll in, or manage different skill-based content.

Profile Management
Personalized user profiles to showcase skills, projects, and achievements.

Opportunity Listing
Access to internships, jobs, or collaborative opportunities based on skills.

Search and Filtering System
Efficient filtering of skills and opportunities based on categories and preferences.

Dashboard Interface
Centralized panel for tracking progress, activities, and user engagement.

Content Management
Users can create, update, and manage their skill-related content securely.

Installation

Clone the repository into your local server directory (e.g., htdocs).

Import the database file (e.g., skillbridge.sql) into MySQL.

Configure database credentials in your connection file (e.g., config/db.php).

Ensure required folders have proper write permissions.

Run the project using XAMPP or Apache server.
